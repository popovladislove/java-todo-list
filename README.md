# Java To-Do List

Консольное приложение на Java для управления списком задач.

## Функциональность
- Добавление задачи
- Просмотр всех задач
- Удаление задачи
- Отметка задачи как выполненной
- Сохранение задач в файл
- Загрузка задач из файла при запуске

## Технологии
- Java
- OOP
- ArrayList
- Scanner
- FileWriter / FileReader / BufferedReader
- Обработка исключений

## Что реализовано
- Класс `Task` с инкапсуляцией данных
- Хранение задач в `ArrayList<Task>`
- Консольное меню через `while` и `switch`
- Разделение логики на отдельные методы
- Сохранение задач в текстовый файл
- Загрузка задач из файла при старте программы
